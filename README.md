# Hastify – Bharatanatyam Mudra Recognition

Hastify is a Computer Vision and Machine Learning project aimed at recognizing Bharatanatyam hand mudras in real time using webcam input. The project is currently under development and focuses on building an interactive learning experience for classical dance enthusiasts.

## Current Progress

* Webcam integration using MediaPipe Hands
* Real-time hand landmark detection
* Recognition of a few basic Bharatanatyam mudras
* React-based user interface

## Tech Stack

* React + Vite
* MediaPipe Hands
* FastAPI (planned backend)
* Scikit-learn (planned ML model)

## Upcoming Features

* Expanded mudra dataset and recognition
* Machine Learning-based classification
* Improved detection accuracy
* Practice and learning mode
* User progress tracking
* Performance analytics dashboard

## Project Status

🚧 Work in Progress

This project is actively being developed. The current version includes the basic application structure, webcam integration, and detection of a limited set of mudras. Additional features and improved AI-based recognition will be added in future updates.
